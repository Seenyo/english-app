# UI copy rules

- Keep all UI text minimal.
- Do not add explanations, translations, helper text, captions, labels, badges,
  or status messages unless the user explicitly requests them or they are
  required to operate the interface.
- Do not replace removed text with alternative wording.
- Prefer clear layout and controls over explanatory copy.
- Accessibility-only labels may be visually hidden.
- When unsure whether visible text is necessary, omit it.
